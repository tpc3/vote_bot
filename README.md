# vote_bot

Vote bot for discord, written with rust.

## Usage(easy)

1. Install rust
2. Open config.toml, fill blanks(bot token, id, encrypt key)
3. Run `cargo run`

## Advanced config

* Change shards value if you have more than 2500 guilds
* Change prefix to change command prefix
